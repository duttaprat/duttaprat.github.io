## ✨ About this site

This is a **single-page academic website** designed to present my research, publications, experience, recognition, and scientific trajectory in a clean, lightweight, and personal format.

The site is built around five core sections:

- 🧠 **Research** — scientific themes, current directions, and long-term vision
- 📰 **News** — recent updates, releases, talks, and milestones
- 📚 **Publications** — selected papers and scholarly contributions
- 🧭 **Experience** — academic and research timeline
- 🏅 **Recognition** — awards, invited talks, peer review, teaching, and mentorship

The goal is simple: to create an academic homepage that feels **distinctive, honest, minimal, and alive** — not just a static CV pasted onto the web.

---

## 🛠️ Tech stack

This site is intentionally simple: **plain HTML, CSS, and JavaScript**.

No build step. No framework. No complicated deployment pipeline.

| Layer | Details |
|---|---|
| Structure | Single `index.html` file |
| Styling | Custom CSS |
| Interactivity | Vanilla JavaScript |
| 3D graphics | [Three.js](https://threejs.org/) loaded from CDN |
| Fonts | Fraunces, Geist, JetBrains Mono, Noto Serif Bengali |
| Hosting | GitHub Pages |

## ⭐ Usage & attribution

If you like this design, you are welcome to fork this repository and adapt it for your own academic website.

If you use this template, I only ask for two things:

1. ⭐ **Star this repository** to show your support.
2. 🔗 **Credit the original design** by keeping a link to [duttaprat.github.io](https://duttaprat.github.io) in your website footer.

### Before using this template

Please make sure to:

- ✍️ **Replace all personal content** — biographies, photos, publication lists, Bengali name rendering, CV, and research descriptions should be your own.
- 🧾 **Maintain the license** — keep the credit comment at the top of `index.html`.
- 🖼️ **Use your own media** — personal photos, screenshots, figures, and CV files are not part of the reusable template content.

The **design, layout, and code** are released under [CC BY 4.0](LICENSE.md).  
Personal content, research text, photographs, publication information, and media remain the property of the original author.
